What Kavach Stores

  • Phone number (for session tracking only)
  • Language preference
  • Transaction amount and recipient (risk scoring)
  • Session state (IDLE / QUESTIONING / RESOLVED)
  • Risk score (numeric)

What Kavach Does NOT Store

Message content is processed in-memory and NOT persisted after session resolves.
  • No call recordings or media
  • No contact list or WhatsApp profile data
  • No location data
  • No Aadhaar / PAN / banking credentials
  • No browsing history or app usage

Trusted Contact Privacy

Trusted contacts receive ONLY: "[Name] may be under pressure from a scammer. Please call them now."
  • They NEVER see conversation content
  • They NEVER see scam message details
  • They NEVER see transaction recipient info
  • They NEVER get access to user's other messages

WhatsApp Integration Scope

  • Permission: message send/receive ONLY
  • No access to: contact list, profile photo, status, groups, media
  • Data flow: User message → Kavach analysis (in-memory) → Response
  • No data shared with any third party

Compliance

  • IT Act 2000 (India)
  • RBI Data Localization Guidelines
  • Digital Personal Data Protection Act (DPDP) 2023
  • All data stored on Indian servers
  • Auto-deletion of session data after 24 hours
